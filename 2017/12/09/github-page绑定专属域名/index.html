之前说在码云上搭建博客，后来发现码云不能绑定域名，于是把博客重新部署在github上，本以为在国外访问会很差，没想到还可以也没有太大的延迟，打算写一下绑定域名的教程，做人做到底，送饭送嘴里。

准备

一个域名
DNSPOD账户
搭好博客的GITHUB

首先说域名，我是在namesilo上买的，很便宜，第一年6刀，续费8刀，这个以后开一篇另讲。

什么是DNS

域名解析服务器，靠它把你要访问的网址找到然后把信息送到你电脑上。

选择DNSPOD原因是因为博客主要是给国内看的，而且国外DNS速度可能会慢而且会被墙。

动手

首先我们要在你自己博客的source 文件夹里创建一个文件CNAME，记住一定要大写，然后把你的域名写上不用加www，然后部署一遍

接着把你的域名的DNS服务器改为DNSPOD的解析服务器

f1g1ns1.dnspod.net
f1g1ns2.dnspod.net

我的域名管理界面是这样的，不一样的域名销售不一样。

其实很简单首先我们注册完DNSPOD以后点击域名解析再点击添加域名

点击进去自己的域名，按如下设置

马赛克的部分为 youname.github.io
就是你部署到github上的页面的访问地址
然后把他们都启用就可以了，等待10分钟，你的github page就能用你的域名访问了
