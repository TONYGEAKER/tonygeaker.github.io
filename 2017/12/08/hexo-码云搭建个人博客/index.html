最近在找资料的时候，看到很多人都搭建了自己的博客，所以有点兴趣，一是为了提升自己的写作，二是为了改变一下习惯给自己做一个记录和总结，至少在大学有点东西，于是在查找资料后发现有两种搭建的方法。

  • VPS + WORDPRESS
  • GITHUB + HEXO

第一种方法需要一个VPS和域名，VPS本来我是想用搬瓦工最低配那个配置进行搭建不过想到那个VPS位置在美国访问可能有点慢，还有就是VPS和域名续费，作为一个学生党当然是选择一个便宜的啦！！

所以我选择了第二种方案。
由于github是国外的，在国内访问的时候可能会有点问题所以我选择 码云+hexo

环境配置

首先我们要配置环境

  • node.js
  • GIT

这两个软件一路点下去就行了。
还有你要注册一个码云的帐号，是用来上传你HEXO的静态网页的。

安装HEXO

打开命令提示符，输入

npm install hexo-cli -g

这里说明一下，NPM是随同NodeJS一起安装的包管理工具，由于NPM的服务器是在国外的，下载安装HEXO在一些地区由于网络原因会出现问题，所以可以选择NPM镜像。

淘宝 npm 地址
临时使用只要在安装前在命令提示符打入如下

npm –registry https://registry.npm.taobao.org install express

安装完后就是这样的(这里稍微用了一下网上的截图由于我已经装好了没办法重现了）

接着说，安装完后输入命令

hexo


显示这样就是安装成功了

搭建

在随便什么地方创建一个文件夹，右键文件夹码，点击Git Bash,会出现一个黑窗口

输入命令：

npm install -g hexo-cli

在输入命令初始化HEXO：

hexo init

打开文件夹可以看一下是否初始化成功

本地搭建

输入命令生成静态文件：

hexo g

成功后启动本地服务器：

hexo s


这样就成功了 在地址栏访问 http://localhost:4000/ 可以了

部署码云

我们先创建一个项目：

然后点击进入你创建的项目

然后修改你的配置文件 _config.yml 在根目录
修改你的deploy 的值并保存


这里根据你个人来修改，URL部分一定要修改不然会出现本地部署成功但是码云上缺失CSS的问题

然后右键文件夹 Git Bash 输入

$ git config –global user.name空格+你的码云的名字
$ git config –global user.email空格+你的码云的邮箱

然后运行

npm install hexo-deployer-git –save

注意这里如果出错或者时间很长也需要用NPM镜像

成功后输入命令进行部署:

hexo deploy

之后会弹出一个输入帐号密码的窗口，输入你注册的码云的帐号和密码就行了
