之前搭完hexo然后部署在github上之后发现百度不能爬取github，查原因之后发现，由于百度的爬虫太频繁已经影响到github的正常运作了，所以就把百度的爬虫给墙了，其他的搜索引擎都可以爬取，为了解决这个问题，再加上优化hexo的初衷，我选择了github+coding双线部署hexo，国内访问coding上的页面，国外访问github上的页面。

准备

首先我们需要coding和github的账户，这个不用多说
我们还需要一个dnspod的账户，这样我们才能把国内的访问全部解析到coding page上
